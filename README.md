# LINKconverter

마이리얼트립 일반 링크를 파트너 링크로 변환하고, 네이버 QR 링크와 QR JPG를 생성하는 Windows용 프로그램입니다.

## 다운로드

오른쪽 Releases에서 최신 버전의 `LINKconverter-*-portable.zip`을 다운로드하세요.

## 사용 방법

1. zip 파일을 다운로드합니다.
2. 압축을 풉니다.
3. `LINKconverter-*-portable.exe`를 실행합니다.
4. 처음 사용 시 `로그인 설정`을 눌러 마이리얼트립 파트너와 네이버에 로그인합니다.
5. 제목과 마이리얼트립 일반 링크를 입력합니다.
6. `시작`을 누릅니다.

## 업데이트

프로그램 안의 `업데이트 확인` 버튼을 눌러 최신 버전을 확인할 수 있습니다.
